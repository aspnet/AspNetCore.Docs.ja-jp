# <a name="aspnet-core-docs-whats-new-for-february-1-2021---february-28-2021"></a>ASP.NET Core docs: 2021 年2月1日の新機能-2021 年2月28日

- <xref:blazor/components/css-isolation> -CSS 分離継承をサポートするためのガイダンスを追加する
