* `-c|--content`

  インライン HTTP 要求本文を提供します。 たとえば、「 `-c "{"id":2,"name":"Cherry"}"` 」のように入力します。

* `-f|--file`

  HTTP 要求本文を含むファイルへのパスを提供します。 たとえば、「 `-f "C:\request.json"` 」のように入力します。

* `--no-body`

  HTTP 要求本文が不要であることを示します。
